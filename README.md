# 🔊 Smart Speaker Project

A multi-language demo project themed around smart speaker technology.

## Features
- Volume control logic (Python)
- Speaker configuration (JSON)
- CLI controller (JavaScript)
- Web UI (HTML/CSS)
- Backend API (Node.js)
- Audio processing simulation (C++)
- Speaker model (Java)
- Sound power calculator (Go)
- Embedded style config (YAML)
- Simple Rust diagnostics tool
